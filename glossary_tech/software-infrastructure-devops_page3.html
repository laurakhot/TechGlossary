Software Infrastructure (DevOps)

Ethernet

Ethernet is a widely used networking technology for connecting devices in local area networks (LANs). 

Exim

EXperimental Internet Mailer is a message transfer software (agent) for Linux and UNIX operating systems. Exim is distributed under the GPL license, and everyone can freely download it, use it and modify it.

Fedora

A Linux/Unix based operating system (Linux distribution) that sponsored by Red Hat and supported by free software community. The project serves to test new technologies, which are further included in the products of Red Hat and other manufacturers.

Filezilla

A multi language, cross-platform file transport application, consisting of FileZilla Client and FileZilla Server. It supports FTP, SFTP and FTPS transport protocols. Equipped with the ability to drag and drop objects, synchronize the directory and search the remote server. Supports multithreaded downloading of files, as well as resume (if supported by the server) on the Internet connection.

Firewalls

A network security systems that monitor incoming/outgoing network traffic, decide whether to allow specific traffic or block it due to security rules, establish a barrier between a secure internal network and Internet.

Fortify

Fortify is a static analysis tool and it processes code in a manner similar to a code compiler. It uses a build tool that runs on a source code file or set of files and converts it into an intermediate model that is optimized for security analysis by Fortify. 

FreeBSD

Unix-like operating system, used to power modern servers, desktops, and embedded platforms, it provides networking, performance, security and compatibility features. This network operating system is an ideal platform for building an Internet or Intranet server.

FTP

File Transfer Protocol is a standard network protocol for transmitting files between computers on the Internet, built on a client-server model architecture. FTP is often used to download web pages and other documents from a private development device to an open hosting server.

Ganglia

A scalable, distributed monitoring tool that provides a complete real-time monitoring and execution environment that allows you to track statistics and history (CPU utilization, network) of real-time calculations for each of the observed nodes.

Gateway

Link between two computer programs or systems, hardware router or software for interfacing computer networks using different protocols. The main task of the network gateway is to convert the protocol between networks. The most common gateway is a router.

GeoServer

GeoServer is a web server that allows you to serve maps and data from a variety of formats to standard clients such as web browsers and desktop GIS programs.

GitOps

GitOps is an operational framework that extends DevOps principles employed in application development, encompassing practices like version control, collaborative work, compliance, and continuous integration/continuous deployment (CI/CD), and applies them to the realm of infrastructure automation.

GNU

A Unix-compatible operating system that uses the Linux kernel. A command line is used to work in this OS, but also a graphical module can be installed for users that don’t work with command line.

Gunicorn

Green Unicorn is a Python Web Server Gateway Interface HTTP Server for UNIX OS. The Gunicorn server is broadly compatible with various web frameworks, simply implemented, light on server resources, and fairly speedy.

HAProxy

High Availability Proxy is a server software to provide high availability and load balancing for TCP and HTTP applications, used to improve the performance of a server environment by distributing the workload across multiple servers.

Hardware installation

The process of installing, configuring and testing a new piece of hardware in a computer. New hardware installed in a computer is system memory, hard drives, video cards, sounds cards, rom drives (CD/DVD/Blue-Ray rom drives), power supplies, and motherboards.

HP Diagnostics

A monitoring tool used to monitor application on different platforms (Java, .Net, Oracle DB, SQL server, SAP, etc), allows to drill down into the code level to find the exact bottleneck and resolve it.

HTTP

HyperText Transfer Protocol is an application protocol for data tranmission in the form of hypertext documents in the "HTML" format, is currently used to transfer arbitrary data and it is the basis for data transfer for the World Wide Web.

https

A protocol for secure communication over an Internet, it consists of Hypertext Transfer Protocol (HTTP) within a connection encrypted by Transport Layer Security (TLS). It provides protection from attacks based on listening to a network connection.

Hybrid Deployment

Hybrid deployment is a strategic approach to managing computer systems and applications. It combines both on-site infrastructure (like private data centers within a company's physical location) and cloud-based services (which are internet-based and provided by external vendors). This approach offers a versatile solution for organizations, allowing them to leverage the benefits of both environments.

Hyper-convergence

A type of infrastructure system with a software-defined architecture, it tightly integrates compute, storage, networking and virtualization resources, frequently used in a data center and by cloud operators.

Icinga

Icinga is an open-source network and system monitoring application that helps organizations keep their IT infrastructure in check. It provides a flexible and extensible platform for monitoring services and hosts, making it easier to detect and respond to issues before they impact operations.

IIS

Internet Information Services is a web server from Microsoft that runs on Windows systems and is used to host websites and other content on the web. IIS supports HTTP, HTTPS, FTP, POP3, SMTP, NNTP protocols.

IM

Instant Messaging is a type of online communications service that provides real-time text transmission over the Internet through a software application, allows to create a private chat room using text-based communication.

IMAP

Internet Message Access Protocol that is a standard email protocol that stores email messages on a mail server, allows to view and manipulate the messages as locally stored on the end user's computing devices.
